# ARIMA Forecast Project

Прогноз временного ряда с моделью AR(2) на второй разности и доверительными интервалами.

## Что внутри
- `time_series.ipynb` — весь код: проверка стационарности, построение модели, прогноз, ДИ.
- `DTBSPCKNMM.xlsx` — исходные данные(Источник: https://fred.stlouisfed.org/series/DTBSPCKNMM)
